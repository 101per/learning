# 今日学习文档：现代视觉模型理解路径

## 一、从 Transformer 进入视觉领域

今天的学习起点是 **Transformer 在视觉中的应用**。

Transformer 最初来自 NLP，但后来被引入视觉领域，形成了
**Vision Transformer**。

核心思想：

```
image
↓
patch 切分
↓
patch embedding
↓
transformer encoder
```

关键变化：

```
CNN：局部卷积
ViT：全局 self-attention
```

这意味着模型可以直接学习 **全局视觉关系**。

------

# 二、Transformer 在目标检测中的应用

接下来研究的是 **Transformer Detection**。

代表模型：

- **DETR**

DETR 的核心思想：

```
image
↓
CNN backbone
↓
transformer encoder-decoder
↓
object queries
↓
prediction set
```

关键创新是：

```
set prediction
```

也就是说：

```
预测一组对象
而不是 anchor-based detection
```

------

## DETR 的关键机制

### 1 Object Query

query 是一组 **可学习向量**：

```
query1
query2
query3
...
queryN
```

每个 query 的任务是：

```
寻找一个物体
```

所以最终输出是：

```
N 个预测框
```

------

### 2 Hungarian Matching

由于预测数量固定，但真实物体数量不固定，需要进行匹配。

方法：

```
prediction ↔ ground truth
```

通过 **匈牙利算法**进行 **最优匹配**：

```
minimize total cost
```

cost 由两部分组成：

```
classification loss
bbox loss
```

------

### 3 bbox（bounding box）

bbox 是：

```
(x_center, y_center, width, height)
```

表示物体在图像中的位置。

------

### 4 background

如果 query 没有匹配到物体：

```
class = background
```

因此 DETR 不需要：

```
anchor
NMS
```

这是和 YOLO 系列的巨大区别。

------

# 三、DETR 的问题与改进

DETR 的主要问题：

```
训练慢
收敛慢
```

原因：

```
attention 需要全局搜索
```

改进模型：

- **Deformable DETR**

核心思想：

```
attention
↓
只关注少量采样点
```

也就是：

```
sparse attention
```

效果：

```
训练速度 ↑
收敛速度 ↑
```

------

# 四、自监督视觉学习

接下来研究的是 **不需要标签的视觉学习**。

代表方法：

- **DINO**

DINO 使用：

```
teacher-student network
```

学习方式：

```
不同图像视角
↓
输出保持一致
```

最终得到：

```
通用视觉表征
```

------

## DINO 的一个惊人现象

训练完成后：

```
ViT attention map
```

会自动变成：

```
object segmentation
```

原因：

模型为了识别物体，需要：

```
自动找到 object boundary
```

这说明：

```
object structure 是视觉特征的一部分
```

------

# 五、视觉基础模型

接下来是视觉领域的重要方向：

```
Foundation Model
```

代表模型：

- **Segment Anything Model**

SAM 的目标：

```
分割任何物体
```

输入：

```
point
box
mask
```

输出：

```
segmentation mask
```

核心思想：

```
promptable segmentation
```

训练数据：

```
SA-1B
1 billion masks
```

这也是 foundation model 成功的重要原因。

------

# 六、视觉语言模型

另一条重要技术路线是 **视觉 + 语言**。

代表模型：

- **CLIP**

CLIP 的核心：

```
image encoder
text encoder
contrastive learning
```

目标：

```
image embedding ≈ text embedding
```

例如：

```
image: dog
text: "a dog"
```

embedding 会接近。

------

## CLIP 的能力

CLIP 可以：

```
zero-shot classification
```

例如：

```
"a photo of a dog"
"a photo of a cat"
```

直接分类。

------

# 七、开放词汇目标检测

CLIP 启发了新的检测方法：

- **GLIP**
- **YOLO-World**

技术发展：

```
CLIP
↓
GLIP
↓
YOLO-World
```

含义：

```
分类 → 检测
```

再进一步：

```
实时检测
```

YOLO-World 实现了：

```
real-time open vocabulary detection
```

------

# 八、现代 YOLO 系列

YOLO 系列目标一直是：

```
real-time detection
```

最新版本：

- YOLOv12

主要变化：

```
CNN detector
+
attention module
```

说明一个趋势：

```
CNN + Transformer
```

正在逐渐融合。
