# 📘 今日学习笔记：视觉 Transformer 与检测

## 1️⃣ Transformer 基础

- **Origin**：`Attention Is All You Need` (2017)，主要用于 NLP。
- 核心思想：**Self-Attention + Token 表示**。
- 不同任务只需调整输入 token 和输出头（head）。

------

## 2️⃣ 视觉 Transformer（Vision Transformer, ViT 系列）

### 主要模型与特点

| 模型 | 特点                                   | 应用/用途                              |
| ---- | -------------------------------------- | -------------------------------------- |
| ViT  | 把图像切成 patch → Transformer encoder | 图像分类，基础 backbone                |
| DeiT | 数据高效训练，引入蒸馏 token           | 小数据集分类，提升训练效率             |
| Swin | 层次化 + Shifted Window Attention      | 多尺度特征，分类、检测、分割 backbone  |
| MAE  | 自监督 masked image modeling           | 提供预训练权重，可迁移到分类/检测/分割 |

------

## 3️⃣ 视觉 Transformer 与目标检测

### 核心思路

- **检测**是视觉 Transformer 的下游任务。
- Backbone（ViT/Swin/MAE）提取特征 → Detection head（DETR/YOLO）预测边界框 + 类别。
- Transformer 在检测中增强 **全局感知能力**，补充 CNN 局部特征。

### 代表性检测 Transformer

| 模型                   | 特点                               | 改进点                        |
| ---------------------- | ---------------------------------- | ----------------------------- |
| DETR                   | 端到端，完全用 Transformer         | 去掉 NMS + RPN，收敛慢        |
| Deformable DETR        | 局部注意力（deformable attention） | 提升收敛速度 & 高分辨率效率   |
| Conditional / DAB-DETR | 条件 query / 动态 anchor           | 降低训练难度，提高效率        |
| Swin-Backbone DETR     | Swin 提供多尺度特征                | 对小目标敏感，适合实际场景    |
| YOLOv12                | 混合 CNN + Transformer             | 保留局部卷积特征 + 全局注意力 |

------

## 4️⃣ Transformer 在检测中的作用与关系

- **ViT / DeiT / Swin / MAE** → backbone（特征提取）
- **MAE** → 自监督预训练，可提升检测性能
- **DETR / YOLOv12** → detection head（边框 & 分类预测）
- **YOLOv12**不是纯 Transformer，仍保留卷积用于局部特征。

### 关系示意（文字版）

```
视觉 Transformer
│
├─ ViT → backbone
├─ DeiT → backbone, 数据高效
├─ Swin → backbone, 局部+多尺度
├─ MAE → backbone预训练
│
└─ Detection Transformer (DETR / YOLOv12)
      ↑
      使用上述 backbone 提取特征
```

------

## 5️⃣ 规律与总结

1. **视觉 Transformer 主要解决全局建模问题**，弥补 CNN 局部感知局限。
2. **检测 Transformer 是下游任务应用**，通常需要 backbone 提供特征。
3. **性能提升手段**：
   - 局部窗口 attention（Swin/Deformable DETR）
   - 自监督预训练（MAE）
   - 条件 query / 动态 anchor（Conditional DETR）
4. **YOLOv12**采用混合结构：卷积 + Transformer → 兼顾局部与全局。
